# Book List App

A simple application for managing a collection of books. Users can browse a list of books, add new titles, and remove books from the collection through an intuitive and easy-to-use interface.

## Features

* Display a list of books
* Add new books dynamically
* Delete books from the list
* Responsive user interface
* Styled using Tailwind CSS
* Built with reusable React components

## Technologies Used

* React
* Vite
* Tailwind CSS
* JavaScript (ES6+)
* Git & GitHub

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Mwas1515/book-list-app.git
```

2. Navigate into the project folder:

```bash
cd book-list-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

##  Build for Production

```bash
npm run build
```
